* Styled Text Input.
* Takes the same attributes that a HTML checkbox does and behaves in the same way.


### React Props
Takes the same props as native text inputs in React.

#### `<Form.TextInput />`
* **`id :String [optional]`**  
Should match the `htmlFor` prop on `<Form.Label>`.
* **`small :bool [optional]`**  
Renders small version.
