Indicates progress.

### React Props

#### `<ProgressBar.Wrapper>`
* `tag :String [optional]`  
Allows the rendered HTML tag to be overridden from the default `<div>`.

#### `<ProgressBar.Indicator>`
* `percent :String [optional]`  
Set the width as a percentge.
* `animated :bool [optional]`
* `tag :String [optional]`  
Allows the rendered HTML tag to be overridden from the default `<div>`.
